HTML

Что это такое?

Это язык разметки, который нужен чтоб отмечать элементы на странице.

К html так же можно обращаться через другие языки.Например Css и JavaScript

Где используеться Html?

Он используеться на сайтах,к примеру ЭТОТ сайт написан на чистом html и ничего больше в нём не используеться,

но как видите он пустой и скучный,для этого и нужны другие языки

Как ты находишь сайты в поисковике?

Для этого в html файле прописываються метаданные и поисковик обращаеться к ним чтоб найты нужный для вас сайт

Пример:meta name="twitter:" content="Котята",если ты будешь искать в поисковике "Котята в твитер",тебе выдаст сайт с этими тегами в meta

Другие языки

Css

Это язык стилистики,его подключают к html.Чтоб добавлять разноцветные кнопки,картинки,анимации и т.д

JavaScript

В свою очередь это уже полноценый язык программирования,на нём обрабатоваются все ваши действия и не смотря на то что анимации можно добавлять через Css для этого чаще используют JavaScript

Интерестный факт: изначально JavaScript был создан толко для разработки сайтов и браузеров,

но позже перерос в многофункциональный язык программирования,который используеться почти везде(Веб приложения,Сайты,Браузеры,И даже иногда используеться в играх!)
